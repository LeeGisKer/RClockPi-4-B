# Weather Sprites

Place PNG files in this folder with these names:

- `clear.png`
- `clear_night.png`
- `mostly_clear.png`
- `partly_cloudy.png`
- `overcast.png`
- `fog.png`
- `drizzle.png`
- `rain.png`
- `snow.png`
- `showers.png`
- `thunder.png`
- `unknown.png`

The Weather page and hourly/weekly forecast will use these sprites based on weather conditions.
If a sprite is missing, the app falls back gracefully.
